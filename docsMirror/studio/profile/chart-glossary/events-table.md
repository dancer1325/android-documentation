#  Events table Stay organized with collections  Save and categorize content based on your preferences. 

The **events table** lists all calls in the selected thread. You can sort them by clicking on the column headers. By selecting a row in the table, you can navigate the timeline to the start and end time of the selected call. This lets you accurately locate events on the timeline.
